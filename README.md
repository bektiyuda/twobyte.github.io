# Three Sword Style CTF Archive ⚔️⚔️⚔️

Website archive write-up CTF dengan tema Roronoa Zoro.

🌐 **Live**: [https://2byte36.github.io](https://2byte36.github.io)

## Quick Start

```bash
npm install
npm run server    # http://localhost:4000
npm run writeup   # CRUD manager
```

## Deployment

Push ke repository, GitHub Actions akan auto-deploy ke GitHub Pages.

## Commands

- `npm run server` - Dev server dengan auto-reload
- `npm run build` - Generate static files
- `npm run clean` - Clean cache
- `npm run writeup` - CRUD manager untuk write-up
